## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/portfolio.git
    ```
2. Navigate to the project directory:
    ```sh
    cd portfolio
    ```
3. Install the dependencies:
    ```sh
    npm install
    ```
